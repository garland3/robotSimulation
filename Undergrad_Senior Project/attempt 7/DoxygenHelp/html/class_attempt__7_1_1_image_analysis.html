Detailed Description

It was easier to redure 640*480 small triangles and view then from a distance than to put a texture on the GPU So there is a camera to view the triangels. The class only analzes a small fraction of the pixels each time through the game loop inorder to keep the speed high. Many of the methods have loops that cause them to only look at every X pixel each time through and then the next time through a different set. Basically it functions like a giant double "for" loop.

The Image Analysis Class does the image processing of the robot view.
