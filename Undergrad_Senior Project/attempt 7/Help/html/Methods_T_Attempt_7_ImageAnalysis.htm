ImageAnalysis Methods
ImageAnalysis Class See Also Send Feedback

The ImageAnalysis type exposes the following members.

Methods

  NameDescription
AddHoughLinesColor
Puts the hough information about the lines in the vertex array.
CalculateStartandStopofLine
Part of the old Hough system. Finds the edge values on the screen of the lines based on the slope and yInt.
ClearMaxInAccum
Clears the array arround the maximum. Gets information about the line to clear around from the 'this.houghInfo' array
Draw
Calls methods to draw the analysis triangles and the Debug text.
(Overrides DrawableGameComponentDraw(GameTime).)
DrawAnalysis
Draws the Analysis Triangles stored in the 'vertexArray'
DrawText
Draws the Text giving important information about the status of the analysis.
FindAverages
Find the average theta values. Weigh them according to the rho value of each one. Make a turning desision based off the weighted thetas if turnBytheta is on.
FindMaxInAccumArrayOfHough
Finds Max value in Hough. Store information about that max.
FindWhite
Takes a 2D color array and finds the pixels that are "white"
GetTurnIndicator
Allows the SimulationMain to get the turnIndicator
Hough
For each white pixel that might be part of a line, Find all the potiential lines going through it and store each vote for that line in the accumlator's bins. Calls the methods to search through the accumlator to find the bins with the largest values.
Initialize
Called when the class is initialized. Creates many of the arrays and sets many of the values.
(Overrides DrawableGameComponentInitialize()()()().)
InsertLine
Updates the vertex information of the triangels for a line of a particular color based off its starting and ending position.
LoadContent
Loads the SpriteBatch Content.
(Overrides DrawableGameComponentLoadContent()()()().)
LoadVertexArray
Loads the position of each triangle into the VertexArray
SetRobotCameraView
Takes a 2D renderTarget/Texture and sets it as the image to analze. Explicitly called in the SimulationMain Draw Method.
ShowPath
Find a path through the map to go through and return it as a bool map, also set the turn indicator. Based off the reactiveNavigation from the old robot code.
Smooth
Takes a truefalse map and for each pixel checks the other pixels around it to see if they are also white. If the number of pixels arround it that are also white is above a threshold then keep that pixel white. Meant to reduce noise in the picture. *******************NOT IN USE RIGHT NOW********************
TextureTo2DArray
Takes a texture and makes it into a 2D color array. Passing in the arrays is faster than trying to build it each time.
Update
Calls the analysis methods that actually do all the work. Basically the Main function for the Image Analysis Class
(Overrides GameComponentUpdate(GameTime).)
Update1
Stores the texture from the robot camera in a color array before the texture is disposed. Expicitly called in the SimulationMain Draw method
UpdateBoolMapto3DRectangle
Colors the analysis triangles based a trueFalse bool map. Also adds the Hough information If true then Blue. If false then Transparent.
UpdateColorArrayto3DRectangle
Takes a 2D color array and colors the triangles. Also adds the hough lines.
Whiteline
Determines if white pixels meet the Width threshold to possibly be a line. *******************NOT IN USE RIGHT NOW********************
