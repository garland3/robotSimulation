Find a path through the map to go through and return it as a bool map, also set the turn indicator. Based off the reactiveNavigation from the old robot code.
Smooth
Takes a truefalse map and for each pixel checks the other pixels around it to see if they are also white. If the number of pixels arround it that are also white is above a threshold then keep that pixel white. Meant to reduce noise in the picture. *******************NOT IN USE RIGHT NOW********************
TextureTo2DArray
Takes a texture and makes it into a 2D color array. Passing in the arrays is faster than trying to build it each time.

Update1

UpdateBoolMapto3DRectangle
